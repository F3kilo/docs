# Drakula API Quickstart

This guide will help you make your first API call to the Drakula platform and start building trading applications.

## Prerequisites

Before you begin, you'll need:

1. **A Drakula account** - [Sign up here](https://app.drakula.com/signup)
2. **API access enabled** - Available on Pro and Enterprise plans
3. **Basic understanding of REST APIs** and your preferred programming language

## Step 1: Get Your API Credentials

### Generate an API Token

1. Log in to your [Drakula Dashboard](https://app.drakula.com/dashboard)
2. Navigate to **Settings** → **API Keys**
3. Click **Generate New API Key**
4. Give your key a descriptive name (e.g., "Trading Bot - Production")
5. Copy the generated token immediately - it won't be shown again!

<Warning>
  **Security Note**: Treat your API token like a password. Never commit it to version control or expose it in client-side code.
</Warning>

## Step 2: Make Your First API Call

Let's test your API credentials by fetching your user profile.

### Using cURL

```bash
curl -X GET \
  "https://api.drakula.com/api/v1/user/profile" \
  -H "Authorization: Bearer YOUR_API_TOKEN_HERE" \
  -H "Content-Type: application/json"
```

### Using Python

```python
import requests

api_token = "YOUR_API_TOKEN_HERE"
base_url = "https://api.drakula.com/api/v1"

headers = {
    "Authorization": f"Bearer {api_token}",
    "Content-Type": "application/json"
}

# Get user profile
response = requests.get(f"{base_url}/user/profile", headers=headers)

if response.status_code == 200:
    user_data = response.json()
    print(f"Welcome, {user_data['data']['username']}!")
    print(f"Your user ID: {user_data['data']['userId']}")
else:
    print(f"Error: {response.status_code}")
    print(response.json())
```

### Using JavaScript/Node.js

```javascript
const fetch = require('node-fetch'); // or use native fetch in newer Node.js

const apiToken = 'YOUR_API_TOKEN_HERE';
const baseUrl = 'https://api.drakula.com/api/v1';

const headers = {
  'Authorization': `Bearer ${apiToken}`,
  'Content-Type': 'application/json'
};

// Get user profile
fetch(`${baseUrl}/user/profile`, { headers })
  .then(response => response.json())
  .then(data => {
    console.log(`Welcome, ${data.data.username}!`);
    console.log(`Your user ID: ${data.data.userId}`);
  })
  .catch(error => console.error('Error:', error));
```

## Step 3: Explore Common Use Cases

### Get Market Data

Fetch token prices and market information:

```python
# Get token details
token_address = "0x...YOUR_TOKEN_ADDRESS..."
chain_name = "ethereum"

response = requests.get(
    f"{base_url}/token/{chain_name}/{token_address}/details",
    headers=headers
)

if response.status_code == 200:
    token_data = response.json()
    print(f"Token: {token_data['data']['tokenSymbol']}")
    print(f"Price: ${token_data['data']['priceUsd']}")
    print(f"Market Cap: ${token_data['data']['marketCapUsd']}")
```

### Execute a Trade

Place a buy order (simplified example):

```python
# Note: This is a simplified example. Real trading requires more parameters.
trade_data = {
    "tokenAddress": "0x...TOKEN_ADDRESS...",
    "amount": "100",  # Amount in USDC or equivalent
    "chain": "ethereum",
    "slippage": 2.0,  # 2% slippage tolerance
    "walletAddress": "0x...YOUR_WALLET..."
}

response = requests.post(
    f"{base_url}/trade/swap/buy",
    headers=headers,
    json=trade_data
)

if response.status_code == 200:
    trade_result = response.json()
    print(f"Trade executed! Trade ID: {trade_result['data']['tradeId']}")
```

### Set Up Copy Trading

Create a copy trading intention:

```python
copy_trade_data = {
    "copyWalletListId": "wallet_list_id_here",
    "amount": {
        "amount": "1000",
        "copyTradeAmountType": "FixedAmount"

        "sellTargets": [
            {
                "sellTargetType": "Multiple",
                "multiple": 2.0,
                "percentage": 50
            }
        ]
    }
}

response = requests.post(
    f"{base_url}/trade/copy-trade/create",
    headers=headers,
    json=copy_trade_data
)
```

## Step 4: Handle Errors Gracefully

Always implement proper error handling:

```python
def make_api_request(endpoint, method="GET", data=None):
    try:
        if method == "GET":
            response = requests.get(f"{base_url}/{endpoint}", headers=headers)
        elif method == "POST":
            response = requests.post(f"{base_url}/{endpoint}", headers=headers, json=data)
        else:
            raise ValueError(f"Unsupported method: {method}")
        
        response.raise_for_status()  # Raise exception for 4xx/5xx responses
        return response.json()
        
    except requests.exceptions.HTTPError as e:
        if e.response.status_code == 401:
            print("Error: Invalid API token. Please check your credentials.")
        elif e.response.status_code == 429:
            print("Error: Rate limit exceeded. Please wait before making more requests.")
        elif e.response.status_code == 400:
            error_data = e.response.json()
            print(f"Validation error: {error_data.get('error', {}).get('message', 'Unknown error')}")
        else:
            print(f"HTTP error: {e}")
        return None
        
    except requests.exceptions.RequestException as e:
        print(f"Request failed: {e}")
        return None

# Example usage
result = make_api_request("user/profile")
if result:
    print("Request successful!")
```

## Step 5: Best Practices

### 1. **Environment Variables**
Store your API token securely:

```bash
# .env file
DRAKULA_API_TOKEN=your_token_here
DRAKULA_API_URL=https://api.drakula.com/api/v1
```

```python
# In your code
import os
from dotenv import load_dotenv

load_dotenv()
api_token = os.getenv("DRAKULA_API_TOKEN")
```

### 2. **Rate Limiting**
- Standard tier: 100 requests/minute
- Implement exponential backoff for retries
- Cache frequently accessed data when possible

### 3. **Webhook Integration**
Set up webhooks for real-time notifications:

```python
# Example webhook handler (Flask)
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/webhooks/trade-update', methods=['POST'])
def handle_trade_update():
    data = request.json
    event_type = data.get('event')
    
    if event_type == 'trade.completed':
        trade_data = data.get('data')
        print(f"Trade completed: {trade_data['tradeId']}")
        print(f"Profit: ${trade_data['profit']}")
    
    return jsonify({"status": "received"}), 200
```

### 4. **Testing**
Use the sandbox environment for testing:

```python
# Sandbox environment (if available)
sandbox_url = "https://sandbox-api.drakula.com/api/v1"
# Use test API tokens for sandbox
```
